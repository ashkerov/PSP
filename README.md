## Цель

Написать функцию `anagram(words)`, которая принимает массив слов и возвращает массив групп анаграмм. Одиночные слова (без пары) отбрасываются.

## Что такое анаграмма

Анаграмма — слова, составленные из одних и тех же букв в разном порядке. Например, `"eat"`, `"tea"` и `"ate"` — анаграммы, потому что все содержат буквы `a`, `e`, `t`.

## Алгоритм

### Ключ — отсортированные буквы

Чтобы определить, являются ли два слова анаграммами, разбиваем каждое слово на буквы, сортируем и склеиваем обратно. Анаграммы дают одинаковый ключ:

```js
const key = word.split("").sort().join("");
// "eat" → ["e","a","t"] → ["a","e","t"] → "aet"
// "tea" → ["t","e","a"] → ["a","e","t"] → "aet"
// "ate" → ["a","t","e"] → ["a","e","t"] → "aet"
```

### Группировка через объект

Используем объект `groups`, где ключ — отсортированные буквы, значение — массив слов с таким ключом:

```js
const groups = {};

for (const word of words) {
  const key = word.split("").sort().join("");
  if (!groups[key]) groups[key] = [];
  groups[key].push(word);
}
// groups = { "aet": ["eat", "tea", "ate"], "ant": ["tan", "nat"], "abt": ["bat"] }
```

### Фильтрация и сортировка

Из всех групп оставляем только те, где >= 2 слов (одиночки не анаграммы). Каждую группу и результат сортируем:

```js
return Object.values(groups)
  .filter((g) => g.length >= 2)    // убираем одиночки ("bat")
  .map((g) => g.sort())            // сортируем внутри группы
  .sort((a, b) => a[0].localeCompare(b[0]));  // сортируем группы между собой
```

## Полный код

```js
function anagram(words) {
  const groups = {};

  for (const word of words) {
    const key = word.split("").sort().join("");
    if (!groups[key]) groups[key] = [];
    groups[key].push(word);
  }

  return Object.values(groups)
    .filter((g) => g.length >= 2)
    .map((g) => g.sort())
    .sort((a, b) => a[0].localeCompare(b[0]));
}

console.log(anagram(["eat", "tea", "tan", "ate", "nat", "bat"]));
```

## Пример работы

Вход:
```
["eat", "tea", "tan", "ate", "nat", "bat"]
```

Шаги:
1. `"eat"` → ключ `"aet"` → groups: `{ aet: ["eat"] }`
2. `"tea"` → ключ `"aet"` → groups: `{ aet: ["eat", "tea"] }`
3. `"tan"` → ключ `"ant"` → groups: `{ aet: ["eat", "tea"], ant: ["tan"] }`
4. `"ate"` → ключ `"aet"` → groups: `{ aet: ["eat", "tea", "ate"], ant: ["tan"] }`
5. `"nat"` → ключ `"ant"` → groups: `{ aet: ["eat", "tea", "ate"], ant: ["tan", "nat"] }`
6. `"bat"` → ключ `"abt"` → groups: `{ aet: [...], ant: [...], abt: ["bat"] }`

Фильтрация (убираем `"bat"`) → сортировка:

Выход:
```
[ ["ate", "eat", "tea"], ["nat", "tan"] ]
```
